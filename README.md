
The code associated with the manuscript entitled **Diet-derived vitamin B12 induces transgenerational inheritance of nematode predation through elevated vitellogenin provisioning** by Quiobe et al. [biorxiv](https://doi.org/10.1101/2025.05.21.655301)

This repository contains:

+ `main_figures.ipynb`: A notebook containing the code to generates the main figures of the manuscript.
+ `Raw phenotyping counts.xlsx` An Excel file containing the Eu mouth-form frequencies for each experimental condition. Each sheet corresponds to a specific genetic mutant or environmental condition, with individual lines (rows) representing replicate measurements and columns representing generations or time points. Refer to the sheet names for the respective experimental background.
* _Note_: stat_inf.zip, which contains the statistical analyses objects generated by PyMC 5.9.1, is required to make the main and the supplementary figures. It can be downloaded from [https://doi.org/10.5281/zenodo.19002703].
